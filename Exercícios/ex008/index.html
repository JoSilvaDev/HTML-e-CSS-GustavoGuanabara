Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag b (não Semântica).

Nesta frase temos um termo em destaque usando Strong ( Semântica, usada para um termo forte).

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag i ( não semântica ).

Nesta Frase temos um termo em ênfase usando a tag em (semântica).

Texto marcado

Podemos também criar um texto marcado usando a tag mark.

Texto grande e pequeno

Podemos também criar textos grandes com a tag big e textos pequenos com a tag small.(Porém, a tag big já esta se tornando obsoleta).

Texto deletado

Podemos marcar um texto como excluído para identificar que ele deve ser lido, mas não considerado.

texto inserido

Podemos marcar um texto inserido para dar uma ênfase e identificar que ele foi adicionado depois.

Existe também a tag u (não semântica), para sublinhar.

Texto sobrescrito

Para inserir coisas do tipo x20+5.

Texto subscrito

Para inserir coisas do tipo H2O.

Texto abreviado

Podemos também inserir um texto abreviado como HTML.
